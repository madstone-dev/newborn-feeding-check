# 신생아 수유 체크 MVP PRD

## 제품 개요

신생아 부모가 하루 수유량과 수유 횟수를 빠르게 기록하고, 현재 몸무게 기준 권장 범위에 들어오는지 확인하는 모바일 우선 정적 웹 서비스다.

## 문제

- 신생아 시기에는 수유량과 수유텀이 빠르게 변해 부모가 적정량을 판단하기 어렵다.
- 수유 기록 앱은 기능이 많아 초기 부모에게 부담이 된다.
- 단순 계산기는 오늘 실제 수유 기록과 연결되지 않아 행동으로 이어지기 어렵다.

## 대상 사용자

- 생후 0~30일 신생아를 돌보는 부모
- 분유 또는 혼합 수유를 하고 있어 ml 단위 기록이 가능한 보호자
- 앱 설치 없이 모바일 브라우저에서 바로 쓰고 싶은 사용자

## MVP 목표

- 현재 몸무게와 목표 수유 횟수로 하루 권장 수유량을 계산한다.
- 오늘 수유 기록을 누적해 총량, 횟수, 평균 1회량을 보여준다.
- 하루 1000ml 상한과 목표 대비 부족/주의 상태를 명확히 표시한다.
- 정적 배포가 가능하고, 광고 영역을 자연스럽게 배치한다.

## 계산 기준

- 하루 권장 수유량: 몸무게 kg x 150~160ml
- 1회 권장 수유량: 하루 권장 수유량 / 목표 수유 횟수
- 기본 목표 수유 횟수: 하루 8회
- 조절 가능 범위: 하루 7~12회
- 분유 하루 상한: 1000ml 이상은 의료진 상담 권장 문구 표시

## 주요 화면

- 오늘 요약: 권장량, 실제 총량, 수유 횟수, 평균 수유량
- 빠른 기록: 최근 권장 1회량을 기준으로 40ml, 60ml, 80ml, 100ml, 직접 입력
- 상태 카드: 목표 전, 권장 범위, 상한 주의, 상담 권장
- 기록 목록: 오늘 수유 기록과 삭제 액션
- 광고 영역: 상단 요약 이후 1개, 하단 1개

## 비범위

- 수면 추적
- 체중 성장 차트
- 모유 직접 수유 시간 기반 추정
- 병원 상담 또는 진단 기능
- 계정, 서버 저장, 보호자 공유

## 안전 문구

- 본 서비스는 기록과 계산 보조 도구이며 의료 상담을 대체하지 않는다.
- 조산아, 저체중아, 황달, 심한 구토, 탈수 의심, 체중 증가 부진은 의료진 상담을 우선한다.
- 생후 첫 며칠은 일반 공식보다 적게 먹을 수 있으므로 아기 상태와 의료진 안내를 우선한다.

## 광고 전략

- MVP에서는 광고 자리만 확보한다.
- 첫 광고는 계산 결과 이후에 배치해 핵심 사용 흐름을 막지 않는다.
- 두 번째 광고는 기록 목록 하단에 배치한다.
- 모바일 기준 320x100, 데스크톱 확장 시 728x90 슬롯을 고려한다.

## 이후 확장

- 생후 일수별 초기 수유량 보정
- 수유텀 알림
- 체중 증가 기록
- 배앓이/토/기저귀 체크
- 50~100일 급성장기 및 분태기 모드
