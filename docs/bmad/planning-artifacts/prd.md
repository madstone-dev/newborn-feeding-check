# 신생아 수유 체크 MVP PRD

## 제품 개요

신생아 부모가 출생일과 몸무게를 기준으로 하루 수유량을 빠르게 계산하고, 오늘 누적 수유량만 간단히 체크하는 모바일 우선 정적 웹 서비스다.

## 문제

- 신생아 시기에는 수유량과 수유텀이 빠르게 변해 부모가 적정량을 판단하기 어렵다.
- 수유 기록 앱은 기능이 많아 초기 부모에게 부담이 된다.
- 단순 계산기는 오늘 실제 수유 기록과 연결되지 않아 행동으로 이어지기 어렵다.

## 대상 사용자

- 생후 0~30일 신생아를 돌보는 부모
- 분유 또는 혼합 수유를 하고 있어 ml 단위 기록이 가능한 보호자
- 앱 설치 없이 모바일 브라우저에서 바로 쓰고 싶은 사용자

## MVP 목표

- 출생일로 생후 일수를 계산하고 초기 수유 주의 문구를 조정한다.
- 현재 몸무게로 오늘 목표 총 수유량을 계산한다.
- 오늘 수유량을 빠르게 더해 총량, 입력 횟수, 남은 최소 목표량을 보여준다.
- 하루 1000ml 분유 상한은 하루 권장량 영역에서 함께 안내한다.
- 정적 배포가 가능하고, 광고 영역을 자연스럽게 배치한다.

## 계산 기준

- 하루 목표 수유량: 몸무게 kg x 150~160ml
- 생후 0~3일: 체중 기준 목표 총량을 표시하되 초기 수유량이 적을 수 있음을 안내
- 생후 4~7일: 체중 기준 목표 총량을 표시하되 빠르게 늘어나는 전환 구간임을 안내
- 생후 8일 이후: 체중 기준 하루 목표 총량을 기본값으로 안내
- 분유 하루 상한: 1000ml 이상은 의료진 상담 권장 문구 표시

## 주요 화면

- 입력: 출생일, 현재 몸무게
- 오늘 요약: 오늘 목표 총량, 실제 총량, 생후 일수, 입력 횟수
- 수유량 추가: 생후 일수별 빠른 선택 버튼, 직접 입력, 추가 버튼
- 오늘 초기화: 잘못 입력했거나 날짜가 바뀌었을 때 오늘 누적 기록 제거
- 광고 영역: 상단 요약 이후 1개, 하단 1개

## 비범위

- 수면 추적
- 체중 성장 차트
- 모유 직접 수유 시간 기반 추정
- 병원 상담 또는 진단 기능
- 계정, 서버 저장, 보호자 공유

## 안전 문구

- 본 서비스는 기록과 계산 보조 도구이며 의료 상담을 대체하지 않는다.
- 조산아, 저체중아, 황달, 심한 구토, 탈수 의심, 체중 증가 부진은 의료진 상담을 우선한다.
- 생후 첫 며칠은 일반 공식보다 적게 먹을 수 있으므로 아기 상태와 의료진 안내를 우선한다.

## 광고 전략

- MVP에서는 광고 자리만 확보한다.
- 첫 광고는 계산 결과 이후에 배치해 핵심 사용 흐름을 막지 않는다.
- 두 번째 광고는 기록 목록 하단에 배치한다.
- 모바일 기준 320x100, 데스크톱 확장 시 728x90 슬롯을 고려한다.

## 이후 확장

- 생후 일수별 초기 수유량 보정
- 수유텀 알림
- 체중 증가 기록
- 배앓이/토/기저귀 체크
- 50~100일 급성장기 및 분태기 모드
